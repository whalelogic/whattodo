# What to Do

#### A helpful AI agent that knows all the secrets to having fun in Connecticut

## Features

- Responses always include location/event cards
- Notebook to save only the things you want
- Easily 'drag and drop' or 'Pin' things to your notebook
- Export or Print your notebook to take with you
- A sleek, vibrant UI that's appealing
- Light/Dark mode toggle
